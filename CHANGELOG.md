# 変更履歴

このプロジェクトの注目すべき変更はすべてこのファイルに記載されます。

形式は [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) に基づいており、
このプロジェクトは [Semantic Versioning](https://semver.org/spec/v2.0.0.html) に準拠しています。

## [3.1.1] - 2026-04-26

### 変更

- **処理プロセスの極限までの最適化**:
  - 文書全体の PMID スキャンを単一パス（1回のみ）に集約し、計算効率を向上。
  - CSL スタイルの解決ロジックを分離・整理し、メインロジックの可読性を改善。
- **既存ドキュメント構造の維持機能を強化**:
  - 既存の参考文献ヘッダー（例：`## 8. References`）を検出した際、そのレベルや章番号を維持したまま内容のみを更新する機能を追加。
- **参考文献リストのクリーニング強化**:
  - 重複するピリオド（`..`）を確実に除去する正規表現処理を導入。

## [3.1.0] - 2026-04-26

### 追加

- **NCBI Literature Citation API への移行**:
  - PubMed `esummary` から公式の `ctxp` エンドポイントへ変更。
  - 最初から完全な CSL-JSON を取得することで、従来のパースエラーを完全に解消。
- **究極の最適化 (Ultra-Optimized Engine)**:
  - スマート・グルーピング機能の実装。
  - オンデマンド・データ処理への移行。

## [3.0.0] - 2026-03-10

### 追加

- **CSL (Citation Style Language) システムへの移行**:
  - `citeproc-py` を導入し、業界標準のスタイル（APA, IEEE 等）に対応。
